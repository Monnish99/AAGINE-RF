# AAGINE-RF: An Interpretable Edge-Aware Hybrid Graph Framework for Neurotoxicity-Related Endpoint Prediction
This study focused on developing a novel hybrid graph neural network for predicting neurotoxicity-related endpoints such as blood-brain barrier (BBB) permeability and neuronal activity (NA). Initially, this study extracts graph embeddings from the novel AAGINE encoder and then pass those extracted features as input to the RF ensemble classifier for the prediction.
# Keywords
Neurotoxicity, Graph neural networks, Attention mechanism, Ensemble learning, Interpretability.
